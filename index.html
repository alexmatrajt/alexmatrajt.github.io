Predicting Customer Churn with Tree-Based Models

Independent Project

Trained a decision tree, random forest, and XGBoost classifier to predict customer churn using the Telco Customer Churn dataset from IBM. Applied class weighting and grid search for hyperparameter tuning to address class imbalance and improve generalization. Evaluated model performance using accuracy, precision, recall, and F1-score. Dog Breed Classifier with Transfer Learning

Ann Arbor, MI
Intro to Machine Learning (EECS 445) - Project
Nov 2024

Built and trained a Convolutional Neural Network to classify dog images as Golden Retrievers or Collies. Faced a shortcut learning issue: Collie images often had green grassy backgrounds, which the model relied on instead of dog features. To address this, used transfer learning by pretraining the model on eight diverse dog breeds to help it learn general dog features. Then fine-tuned different numbers of layers on the target dataset to find the best out-of-sample accuracy—transfer learning significantly improved performance and reduced background bias.
